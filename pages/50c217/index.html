Arthas（阿尔萨斯）Java诊断工具_浅入了解

# Arthas

# Arthas是啥子东西？

Arthas是阿里开源的java诊断工具，我觉得很好用。

我常常用它来解决一下问题：

  1. wo尼玛，我代码怎么没有执行？
  2. wo尼玛，这个方法的入参是啥？
  3. wo尼玛，这个方法返回了个啥？
  4. wo尼玛，这个东西功能这么强！

Arthas支持JDK 6+，支持Linux/Mac/Windows，采用命令行交互模式，同时提供丰富的 Tab 自动补全功能，进一步方便进行问题的定位和诊断。 这句话Copy的

# 安装

# 快速安装arthas-boot（官方推荐）

我一般是在Docker里面使用，

  • 先这样 先进入容器内部
  • 然后这样 使用curl下载arthas-boot.jar的jar包
  • 再这样 然后用java -jar的方式启动

# 一顿操作

docker exec -it 容器ID bash/sh
curl -O https://arthas.aliyun.com/arthas-boot.jar
java -jar arthas-boot.jar

# 选择Java进程

这个时候会提示你找到哪些Java进程，如果有多个进程请输入序号，如果只有一个直接回车就完事了

[INFO] arthas-boot version: 3.4.1
[INFO] Found existing java process, please choose one and input the serial number of the process, eg : 1. Then hit ENTER.
* [1]: 1 /mrtao-docker-file.jar

# 启动完成

因为我是在容器内部启动的，只有一个程序，所以直接回车！
