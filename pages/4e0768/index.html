# Swagger 接入 YApi

# 1.同步接口文档

  1. 根据Yapi项目同步，这里以库存为例

  2. 找到swagger的分组

  3. 并使用开发者模式找到这个分组的请求信息

    4.复制路径到YApi的项目 -> 数据管理

    http://192.168.0.35:8001/songbaisys/v2/api-docs?group=kucun

    • 如果本身存在数据选择智能合并

# 2.层级对应关系

YApi Swagger 代码
项目：库存 分组：inventory 配置文件：com.ccdental.config.Swagger2Config#inventoryDocket
------分类：入库单 ------分类：入库单 @Api(tags = "入库单")
------------接口：新建入库单 ------------接口：新建入库单 @ApiOperation(value = "新建入库单及商品"）

# 3.新建Swagger分组

  1. 在swagger配置文件下新增 Docket

    配置文件路径：com.ccdental.config.Swagger2Config

        //库存
    @Bean
    public Docket inventoryDocket() {
        return getDocket(
                "inventory",   //groupName 分组名称，不能用中文
                "com.ccdental",  //基础扫描路径，swagger将会扫描这个包下的接口
                "库存",
                "库存相关接口",
                "lt",   //简介
                new String[]{    //路径过滤，将会过滤改接口地址下的所有接口并生成文档
                        "/inventory/.*",
                        "/warehousingentry/.*",
                        "/outboundorder/.*",
                        "/warehouse/.*",
                        "/warehousingmove/.*",
                        "/appComposeSplitOrder/.*",
                        "/inventorycheck/.*",
                        "/inventorydamage/.*",
                        "/inventoryspill/.*",
                        "/sncommodity/.*",
                        "/sncommodity/.*",
                        "/outboundreview/.*",
                        "/medicaldevicemaintenancerecord/.*",
                        "/new/warehousingentry/.*",
        }
        );
    }
